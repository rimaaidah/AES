# AES-128 Simulator

Project web simulasi AES-128 dibuat dari nol dengan HTML5, CSS3, dan Vanilla JavaScript ES6 Module.

## Fitur
- Enkripsi dan dekripsi AES-128 1 blok
- Key Expansion RK0 sampai RK10
- State Matrix 4x4 untuk setiap operasi
- Dark mode dan light mode
- Responsive desktop, tablet, mobile
- Accordion, toast, spinner, copy result

## Cara menjalankan
1. Buka folder project di VS Code.
2. Jalankan dengan Live Server.
3. Isi plaintext dan key.
4. Klik Encrypt atau Decrypt.

## Catatan
- Implementasi AES native, tanpa CryptoJS atau library kriptografi lain.
- Gunakan input sample:
  - Plaintext: 00112233445566778899aabbccddeeff
  - Key: 000102030405060708090a0b0c0d0e0f
